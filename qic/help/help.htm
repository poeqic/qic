QIC Search Help

Commands

reload
Reloads overlay_config.ini and the GUI.
searchexit or sexit
Stops QIC. You'll need to run qic.ahk again to execute commands.
0,1,2..n
Generate WTB message for item #n and paste to chat (You have to send it yourself).
swtb{n}
Saves WTB message #n to savedWTB_messages.txt.
who{n}
Sends /whois for seller of item #n.
swtb{n}
Saves WTB message #n to savedWTB_messages.txt.
sort*
Sort current results (see keywords/sort.txt).
view{n}
View all stats for item #n.
page{n}
Jump to search result page #n
listleagues
Displays a List of all available Leagues you can search in.
setleague{n}
Writes league #n to config file.
setps{n}
Writes PageSize #n to config file.

Shortcuts

ctrl+q
Toggle the GUI ON/OFF.
PageUp / PageDown
Browse through search results.
